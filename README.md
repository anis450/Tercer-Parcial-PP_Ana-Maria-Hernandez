# Tercer Final - Paradigmas de Programación

Este repositorio contiene el desarrollo completo de los tres puntos solicitados en el parcial final de la asignatura **Paradigmas de Programación**. Cada punto está organizado en su propia carpeta y documentado de manera independiente.

##  Estructura del Repositorio

```
/README.md              ← Este archivo
/punto1/                ← Diseño con Concurrencia y Cálculo π
/punto2/                ← Diseño usando el paradigma de Aspectos (AOP)
/punto3/                ← Implementación en RUST y comparación con Python
```

---

##  Punto 1: Concurrencia + Cálculo PI

Diseño detallado de una solución de regresión lineal utilizando el paradigma de concurrencia y el cálculo π (pi‑calculus). Incluye:

* Arquitectura por procesos.
* Definición de canales.
* Flujo comunicacional.
* Modelo formal.
* Sincronización y propiedades.

 Archivo: `punto1/README.md`

---

##  Punto 2: Paradigma de Aspectos (AOP)

Diseño de una solución aplicando programación orientada a aspectos. Incluye:

* Identificación de cross‑cutting concerns.
* Definición de aspectos, joinpoints y advices.
* Flujo del sistema con aspectos.

 Archivo: `punto2/README.md`

---

##  Punto 3: Implementación en RUST + Comparación con Python

Incluye:

* Implementación de regresión lineal en Rust.
* Implementación equivalente en Python.
* Pruebas de rendimiento.
* Comparación analítica.

 Carpeta: `punto3/`
